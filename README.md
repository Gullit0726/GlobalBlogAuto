## 🔧 설치 및 실행
```bash
# 1. 저장소 클론
git clone https://github.com/your-repo/global-blog-automation

# 2. 의존성 설치
npm install
pip install -r requirements.txt

# 3. 환경 변수 설정
cp .env.example .env

# 4. 데이터베이스 초기화

# 5. 개발 서버 실행
npm run dev

# 6. Vercel 배포
vercel deploy
```
